## 🚀 Quick Start

```bash
# Terminal 1: Start Sanity Studio
npm run sanity

# Terminal 2: Start Next.js
npm run dev

# Test connection
npm run test-sanity
```

Visit:
- Studio: `http://localhost:3333`
- Website: `http://localhost:3000/shows`
